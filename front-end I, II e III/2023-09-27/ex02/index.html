Revisão Módulo Front-End I

O que é desenvolvimento Front-End?

Desenvolvimento Front-End pode ser classificada como a programação da parte visual do desenvolvimento para web, englobando a construção de interface gráfica, interação e ações no lado do cliente de sites e web apps.

Mesmo sendo constantemente confundido com o design, tratam-se de áreas diferentes, enquanto o designer vai usar ferramentas visuais para desenhar as intefaces, o desenvolvedor Front-End estará mais próxima do código em si, que irá rodar num navegador Web, como Chrome, Firefox, etc.

O que faz o desenvolvedor Front-End?

Entre as funções do desenvolvedor Front-End estão: a criação de layouts responsivos, semânticos e acessíveis; a programação das interações do usuário no Client-Side; e, a programação de recursos de ranqueamento em mecanismos de busca e pesquisa, como o Google
