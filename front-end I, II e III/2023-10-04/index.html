Dicionário de Termos:

Algoritmo
É como chamamos o passo-a-passo de comandos dados ao computador. Utiliza variáveis e controles de fluxo para determinar comportamentos.
Ex.: Acessar uma página com login
API - Application Programming Interface
Em português, interface de programação de aplicativos. É uma interface que disponibiliza serviços para aplicativos.
Ex.: API do Google Maps é usado pelo UBER
Bug
É como chamamos os erros que ocorrem em códigos ou aplicações. Geralmente acontece algum comportamento inesperado e que precisa ser corrigido.
Ex.: Tela azul de erros no Windows
Codar
Tem o mesmo significado que “programar” para os desenvolvedores.
Ex.: “Vou codar hoje” = “Vou programar hoje”
Commitar
É um mecanismo que “salva” o estado atual do projeto utilizando o GIT, escrevendo um comentário e salvando a versão.
Ex.: “Vou voltar para o commit anterior que estava funcionando”
Debug / Debugar
É como chamamos o processo final de uma aplicação ou site que estará online. Clientes e consumidores já podem visitar e utilizar.
Ex.: “Hoje faremos o deploy da nova versão”
Framework
É um pacote de códigos prontos que pode ser usado para criar aplicações. O uso do framework correto otimiza o tempo de desenvolvimento.
Ex.: “Meu framework favorito é o React!”
